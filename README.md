🚀 Career Pathfinder (AI-Based Career Recommendation System)

Career Pathfinder is a full-stack web application that helps users
discover the best career paths based on their skills, interests, and
profile data using Machine Learning.

------------------------------------------------------------------------

🌟 Features

-   User Authentication (Register / Login / Logout)
-   Profile Management (Edit profile + upload profile picture)
-   AI Career Prediction System
-   Career Recommendation with Suitability Score
-   Career History Tracking
-   Explore Roles Section
-   Skill Management System

------------------------------------------------------------------------

🛠 Tech Stack

Frontend: HTML, CSS, JavaScript, Bootstrap
Backend: Python, Flask
Database: SQLite
ML: Scikit-learn, Pandas, NumPy

------------------------------------------------------------------------

📂 Project Structure

Career-Pathfinder/ │ ├── static/ ├── templates/ ├── users.db ├── app.py
├── predict_advanced.py └── README.md

------------------------------------------------------------------------

⚙️ Setup

1.  Clone repo: git clone
    https://github.com/your-username/career-pathfinder.git

2.  Install dependencies: pip install -r requirements.txt

3.  Run app: python app.py

4.  Open: http://127.0.0.1:5000/

------------------------------------------------------------------------

🤖 How It Works

User inputs skills → ML model processes → returns best career paths with
scores.
